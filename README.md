# Phishing Email Detection System

## Description
This project is a Python-based phishing email detection system that analyzes email content to identify phishing attempts. It detects suspicious keywords, extracts and analyzes malicious URLs using Regular Expressions (Regex), and applies a rule-based phishing risk scoring mechanism to classify emails as High, Medium, or Low risk.

## Features
- Keyword-based phishing detection
- Malicious URL detection using Regex
- Rule-based phishing risk scoring
- Email classification (High / Medium / Low)
- Tested with phishing and legitimate emails

## Project Structure
- phishing_detector.py → Main Python script
- sample_emails/ → Contains phishing and safe email samples
- ppt/ → Project presentation
- screenshots/ → Output screenshots

## Tools Used
- Python
- Regular Expressions (Regex)
- Visual Studio Code

## Output
The system successfully detects phishing indicators and classifies email risk levels accurately.
